# Reminders
A CLI Tool for Reminders made using GoLang!

#### Made using 
- [Go](https://go.dev/)
- [Flag Package](https://pkg.go.dev/flag)
- [Node Notifier](https://www.npmjs.com/package/node-notifier)

1. Clone the repo
```sh
git clone https://github.com/vidhik2002/Reminders.git
```
2. Move into folder
```sh
cd Reminders
```
3. Run the command
```sh
make client
./bin/client --help
```
